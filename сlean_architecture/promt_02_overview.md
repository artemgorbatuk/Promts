# Введение и архитектура проекта

Этот документ описывает ключевые принципы проекта: архитектуру, используемые технологии и структуру каталогов.

### Технологии и паттерны

- Архитектура: Clean Architecture с разделением на слои
- ORM: Entity Framework Core с PostgreSQL
- DI: Встроенная система внедрения зависимостей
- Паттерны: Repository, Unit Of Work
- Соглашения: все тексты выносятся в static-классы, сервисы возвращают ResponseInfo

### Структура проекта

```
src/ProjectName/
├── Datasource.Ef/        # Слой данных
│   ├── Models/           ## Модели Entity Framework
│   ├── Configurations/   ## Конфигурации EF
│   ├── Contexts/         ## DbContext
│   └── Migrations/       ## Миграции БД
├── Repositories.Ef/      # Слой репозиториев
│   ├── Api/              ## Интерфейсы и реализации
│   └── Options/          ## Опции запросов
├── Services/             # Слой сервисов
│   ├── Api/              ## Интерфейсы и реализации сервисов
│   ├── Models/           ## Модели запросов/ответов
│   └── Texts/            ## Текстовые константы
└── Client/               # Слой представления
    ├── Controllers/      ## Контроллеры MVC
    └── Views/            ## Razor представления
```
